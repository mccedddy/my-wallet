# My Wallet #

**My Wallet** is a web application designed to help users track their wallets more efficiently. Users can create multiple wallets, record their values, and visualize how their wallets grow over time through graph.

I developed "My Wallet" for personal use because I wanted to keep track of my financial data efficiently.

## Features ##
- **Add and Manage Wallets:** Easily create, edit, and delete wallets.
- **Track Wallet Values:** Record and update values for each wallet.
- **Graphical Data Visualization:** View your wallet growth over time with dynamic charts.
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens.

## Tech Stack ##
### Frontend ###
- **ReactJS**
- **Tailwind CSS**

### Backend & Database ###
- **Firebase Firestore**
- **Firebase Hosting**
